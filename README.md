# wheather-app-streamlit



## Features

- 5 day weather forecast
- Impending Weather changes
- Weather Graph
- Sunrise and sunset times
- CLoud Coverage
- Wind Speed

## Built Using

- Python
- Streamlit
- Plotly
- Python Open Weather Map API
